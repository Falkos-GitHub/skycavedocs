# Sonstige Events

Neben den wiederkehrenden Events wie dem `/vote event` und dem Abbau Event gibt es einige Events, die zu passendem Anlass vom Team durchgeführt werden. Davor gibt es in der Regel eine Ankündigung im Discord oder im Ingame Chat. Dazu gehören folgende Events:

* Quiz Event (`/warp quiz` oder `/warp quiz2`)
* Jump and Run Event (`/warp jump`)
* Spleef (`/warp spleef`)
* Skin Wettbewerb (`/warp skin`)
* Lava Event (Beitritt per `/lavaevent join`)
